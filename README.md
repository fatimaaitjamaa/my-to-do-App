# My First Python To-Do List App 

A simple command-line application to manage your daily tasks.

### Features:
- *Add Tasks:* Create new tasks easily.
- *View Tasks:* See your tasks list with numbers.
- *Delete Tasks:* Remove tasks by their number (Index).
- *Auto-Save:* All tasks are saved in a tasks.txt file.
